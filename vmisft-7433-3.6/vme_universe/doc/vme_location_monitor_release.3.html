NAME

vme_location_monitor_release - Releases the VMEbus register access image.

SYNOPSIS

vme_location_monitor_release ( vme_bus_handle_t bus_handle, vme_lm_handle_t monitor_handle )

PARAMETERS

vme_bus_handle_t bus_handle

Handle allocated by vme_init.

vme_lm_handle_t monitor_handle

Window handle allocated by vme_location_monitor_create.

DESCRIPTION

Releases the VMEbus register access image.

SEE ALSO

vme_location_monitor_create(3)

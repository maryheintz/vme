NAME

vme_register_image_release - Disables the VMEbus register access image.

SYNOPSIS

vme_register_image_release ( vme_bus_handle_t bus_handle, vme_vrai_handle_t window_handle )

PARAMETERS

vme_bus_handle_t bus_handle

Handle allocated by vme_init.

vme_vrai_handle_t window_handle

Window handle allocated by vme_register_image_create.

DESCRIPTION

Disables the VMEbus register access image.

SEE ALSO

vme_register_image_create(3)

NAME

vme_interrupt_release - Uninstalls an interrupt handler.

SYNOPSIS

int vme_interrupt_release ( vme_bus_handle_t bus_handle, vme_interrupt_handle_t interrupt_handle )

PARAMETERS

vme_bus_handle_t bus_handle

Handle allocated by vme_init.

vme_interrupt_handle_t interrupt_handle

Interrupt handle allocated by vme_interrupt_attach.

DESCRIPTION

Uninstalls an interrupt handler.

RETURNS

0 on success, or -1 on failure.

SEE ALSO

vme_interrupt_attach(3), vme_interrupt_generate(3)

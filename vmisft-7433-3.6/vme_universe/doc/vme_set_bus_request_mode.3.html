NAME

vme_set_bus_request_mode - Sets the VMEbus request mode.

SYNOPSIS

int vme_set_bus_request_mode ( vme_bus_handle_t bus_handle, int br_mode )

PARAMETERS

vme_bus_handle_t bus_handle

Handle allocated by vme_init.

int br_mode

Bus request mode. Valid values are:

- VME_DEMAND
- VME_FAIR

DESCRIPTION

Sets the VMEbus request mode.

RETURNS

0 on success, or -1 on failure.

SEE ALSO

vme_get_bus_request_mode(3)

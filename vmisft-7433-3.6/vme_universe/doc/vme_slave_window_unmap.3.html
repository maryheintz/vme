NAME

vme_slave_window_unmap - Unmaps the VMEbus Slave Window.

SYNOPSIS

int vme_slave_window_unmap ( vme_bus_handle_t bus_handle, vme_slave_handle_t window_handle )

PARAMETERS

vme_bus_handle_t bus_handle

Handle allocated by vme_init.

vme_slave_handle_t window_handle

Window handle previously allocated by vme_slave_window_create.

DESCRIPTION

Unmaps the VMEbus Slave Window.

RETURNS

0 on success, or -1 on failure.

SEE ALSO

vme_slave_window_create(3), vme_slave_window_map(3), vme_slave_window_phys_addr(3), vme_slave_window_release(3)

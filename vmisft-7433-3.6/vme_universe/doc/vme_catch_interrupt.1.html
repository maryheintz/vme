NAME

vme_catch_interrupt - Sets up an interrupt handler to retrieve VMEbus interrupts.

SYNOPSIS

vme_catch_interrupt [-f flags] [-l level] [-v vector]

OPTIONS

-f flags

Flags to be passed to the vme_interrupt_attach function. Possible values are:

- VME_INTERRUPT_SIGEVENT - Pass a sigevent struct to vme_interrupt_attach
- VME_INTERRUPT_BLOCKING - vme_interrupt_attach blocks (default)
-l level

VMEbus interrupt level. Possible values are:

- VME_INTERRUPT_VOWN
- VME_INTERRUPT_VIRQ1
- VME_INTERRUPT_VIRQ2
- VME_INTERRUPT_VIRQ3
- VME_INTERRUPT_VIRQ4
- VME_INTERRUPT_VIRQ5
- VME_INTERRUPT_VIRQ6
- VME_INTERRUPT_VIRQ7
- VME_INTERRUPT_DMA
- VME_INTERRUPT_LERR
- VME_INTERRUPT_VERR
- VME_INTERRUPT_SW_IACK
- VME_INTERRUPT_SW_INT
- VME_INTERRUPT_SYSFAIL
- VME_INTERRUPT_ACFAIL
- VME_INTERRUPT_MBOX0
- VME_INTERRUPT_MBOX1
- VME_INTERRUPT_MBOX2
- VME_INTERRUPT_MBOX3
- VME_INTERRUPT_LM0
- VME_INTERRUPT_LM1
- VME_INTERRUPT_LM2
- VME_INTERRUPT_LM3
-v vector

VMEbus interrupt vector

DESCRIPTION

Sets up an interrupt handler to retrieve VMEbus interrupts.

SEE ALSO

vme_generate_interrupt(1)

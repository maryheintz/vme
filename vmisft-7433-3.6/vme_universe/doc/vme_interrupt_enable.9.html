NAME

vme_interrupt_enable - Enable a VMEbus interrupt. Not applicable with Solaris.

SYNOPSIS

int vme_interrupt_enable ( vme_bus_handle_t bus_handle, vme_interrupt_handle_t interrupt_handle )

PARAMETERS

vme_bus_handle_t bus_handle

Handle allocated by vme_init.

vme_interrupt_handle_t interrupt_handle

Handle allocated by vme_interrupt_attach.

DESCRIPTION

Enable a VMEbus interrupt.

RETURNS

0 on success, or -1 on failure.

NOTES

This function is olny available from within the Linux kernel.

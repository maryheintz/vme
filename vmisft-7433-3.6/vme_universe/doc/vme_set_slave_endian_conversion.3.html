NAME

vme_set_slave_endian_conversion - Sets the Slave Window endian conversion feature on or off.

SYNOPSIS

int vme_set_slave_endian_conversion ( vme_bus_handle_t bus_handle, int endian )

PARAMETERS

vme_bus_handle_t bus_handle

Handle allocated by vme_init.

int endian

A value of 1 turns on endian conversion. A value of 0 turns off endian conversion.

DESCRIPTION

Sets the Slave Window endian conversion feature on or off.

RETURNS

0 on success, or -1 on failure.

vme_get_endian_conversion_bypass(3), vme_get_master_endian_conversion(3), vme_get_slave_endian_conversion(3), vme_set_endian_conversion_bypass(3), vme_set_master_endian_conversion(3)

NAME

vme_dma_buffer_unmap - Not used.

SYNOPSIS

int vme_dma_buffer_unmap ( vme_bus_handle_t bus_handle, vme_dma_handle_t handle )

PARAMETERS

vme_bus_handle_t bus_handle

Handle allocated by vme_master_init.

vme_dma_handle_t handle

Handle to a buffer allocated by vme_dma_buffer_create.

DESCRIPTION

Not used. Return 0;

RETURNS

Always return 0.

SEE ALSO

vme_dma_buffer_create(3), vme_dma_buffer_map(3), vme_dma_buffer_phys_addr(3), vme_dma_buffer_release(3), vme_dma_read(3), vme_dma_write(3)
